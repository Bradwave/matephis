# Testing the Drawing Feature

This page is for testing the new drawing feature.

1. You should see a stylus icon in the top right.
2. Clicking it should open the drawing toolbar.
3. You should be able to draw on the page.
4. Scrolling should be locked when drawing is enabled.
5. Eraser and pen sizing should work.
6. The lock button should toggle scroll lock.
